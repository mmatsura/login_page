# Login & Registration System (PHP)

# Project Description

This is an user authentication system built using HTML, CSS, JavaScript, and PHP.

The project includes:

- Login form
- Registration form
- PHP form handling
- Basic backend logic
- Responsive design

# Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## How to Run

1. Install a local server (XAMPP / WAMP)
2. Move the project folder to `htdocs`
3. Start Apache and MySQL
4. Open in browser:  
   http://localhost/LOGIN_PAGE
